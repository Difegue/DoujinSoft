There are several ways to share your content:
  • filter_1(3)DS + GodMode9i
    If you have a flash cart or a modded DSi or 3DS, and you're playing the game using a cartridge, you can use GodMode9i to extract the savefile. Use WarioWare DIY Editor, a program written in Java, to dump the .mio files from the savefile.
  • filter_2(3)DS + Wiimmfi
    If you have Wiimmfi set up, you can upload your content using the Warehouse feature of the game. All content uploaded to Wiimmfi is grabbed periodically and added to the site. Note that you can only have 2 games, 2 records, and 2 comics uploaded at a time.
  • filter_3Wii + RiiConnect24
    If you have RiiConnect24 set up with DoujinSoft, you can simply send your games with DoujinSoft as the recipient. Once we receive it, you will get a message sent to your Wii within an hour confirming it's been added.
  • filter_4Wii + Save File
    If your Wii's modded, you can use SaveGame Manager GX to copy your savefile to your SD Card or USB drive. Use WarioWare DIY Editor, a program written in Java, to dump the .mio files from the savefile.
  • filter_5melonDS
    If you use melonDS, a DS emulator, the savefile for the game will be in the same folder as the ROM. Save it as a Raw Save File. You will need to expand the savefile to be twice the size (33554432 bytes), which can be done with a hex ditor or maybe a text editor. Use WarioWare DIY Editor, a program written in Java, to dump the .mio files from the savefile.
  • filter_6DeSmuME
    If you use DeSmuME, you can go to File > Export ROM Save File... to extract the savefile. Save it as a Raw Save File. You will need to expand the savefile to be twice the size (33554432 bytes), which can be done with a hex editor or maybe a text editor. Use WarioWare DIY Editor, a program written in Java, to dump the .mio files from the savefile.
